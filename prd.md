# Project Requirement Document (PRD): HK Trading Corporate Site

## 1. Project Overview
We are building a professional, responsive corporate website for a local Hong Kong trading company named "Harbour Trade Co.".
The goal is to establish trust, showcase products/services, and generate leads via a contact form.

## 2. Tech Stack (Strict)
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Icons:** Lucide React
- **UI Components:** shadcn/ui (recommended for speed and aesthetics)
- **Deployment:** Vercel compatible

## 3. Core Features
1.  **Bilingual Support:** - The site must support Traditional Chinese (繁體中文) and English.
    - A language switcher toggle in the Navbar.
2.  **Responsive Design:** Mobile-first approach. Looks good on iPhone and Desktop.
3.  **Navigation:** Home, About Us, Services, Contact.

## 4. Page Structure & Content
### A. Navbar
- Logo (Left)
- Links: Home, About, Services, Contact (Center/Right)
- Language Switcher (Right)
- "Get a Quote" CTA Button (Right)

### B. Home Page (Hero Section)
- High-quality background image or clean gradient.
- Headline: "Connecting Global Markets with Hong Kong Efficiency."
- Sub-headline: "Your trusted partner in import/export solutions since 2010."
- CTA Buttons: "Our Services" / "Contact Us"

### C. About Us Page
- Company History
- Mission & Vision
- "Why Choose Us" section (3-4 key selling points with icons).

### D. Services/Products Page
- Grid layout displaying 3 core services:
  1. Sourcing & Procurement
  2. Logistics & Shipping
  3. Quality Inspection
- Each card has an icon, title, and short description.

### E. Contact Page
- Contact Form (Name, Email, Phone, Message).
- Company Info (Address in Kowloon Bay, Phone, Email).
- Embed Google Map placeholder.

## 5. Design Aesthetic
- **Vibe:** Professional, Trustworthy, Clean, Corporate.
- **Color Palette:** - Primary: Deep Navy Blue (Trust)
  - Secondary: Gold/Amber (Premium/Trade)
  - Background: White/Light Gray
